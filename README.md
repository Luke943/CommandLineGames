## Description

A collection of short games made to run on the command line.

### Contents
- Blackjack (`Python`)
- Hangman (`C`)
- Rock, Paper, Scissors (`JavaScript`)
- Rock, Paper, Scissors, Lizard, Spock (`C++`)
- Tetris (`C++`)
- TicTacToe (`Python`)
- UFO game (`C++`)
